CASE Statements

جمل وتعبيرات CASE

🎯 متى نستخدم CASE؟

تستخدم CASE عندما يكون لديك انشعابات شرطية كثيرة (بدائل متعددة). وهي غالباً أكثر ترتيباً ووضوحاً من IF-ELSIF المتكررة.

🔄 الأنواع

النوع الوصف الاستخدام
Simple CASE Statement تقارن متغير واحد بقيم متعددة. عندما تريد فحص المساواة (=) لمتغير واحد.
Searched CASE Statement تقيم شروطاً مستقلة معقدة. عندما تكون الشروط معقدة (>, <, IS NULL).
CASE Expression تُرجع قيمة واحدة (تستخدم في الإسناد). v_res := CASE ... END;

💻 أمثلة للكتابة

1. Simple CASE (فحص المساواة)

CASE v_grade
    WHEN 'A' THEN DBMS_OUTPUT.PUT_LINE('Excellent');
    WHEN 'B' THEN DBMS_OUTPUT.PUT_LINE('Very Good');
    ELSE DBMS_OUTPUT.PUT_LINE('Good');
END CASE;

2. Searched CASE (شروط مختلفة)

CASE
    WHEN v_salary > 20000 THEN v_tax := 0.2;
    WHEN v_salary BETWEEN 10000 AND 20000 THEN v_tax := 0.15;
    ELSE v_tax := 0.1;
END CASE;

3. CASE Expression (ترجع قيمة)

v_status := CASE v_grade
                WHEN 'A' THEN 'Excellent'
                ELSE 'Standard'
            END; -- لاحظ: لا يوجد END CASE هنا، فقط END

⚠️ ملاحظة هامة

في CASE Statement، إذا لم يتحقق أي شرط ولم يكن هناك ELSE، سيحدث خطأ: CASE_NOT_FOUND_EXCEPTION.
