Declaring PL/SQL Variables

تعريف المتغيرات في PL/SQL

📖 استخدامات المتغيرات | Use of Variables

المتغيرات تُستخدم لـ:

  • التخزين المؤقت للبيانات (Temporary Storage)
  • معالجة القيم المخزنة (Manipulation of Stored Values)
  • إعادة الاستخدام (Reusability)
-- مثال: تخزين بيانات من جدول في متغيرات
SELECT first_name, department_id
INTO   emp_fname, emp_deptno
FROM   employees
WHERE  employee_id = 100;

-- emp_fname = 'Jennifer'
-- emp_deptno = 10

🏷️ قواعد تسمية المعرّفات | Identifiers

عند تسمية المتغيرات، يجب مراعاة:

  • يجب أن يبدأ بـ حرف (letter)
  • يمكن أن يحتوي على حروف أو أرقام
  • يمكن أن يحتوي على رموز خاصة: $ _ #
  • الحد الأقصى للطول: 30 حرفاً
  • لا يمكن أن يكون كلمة محجوزة (Reserved Word)

⚙️ الصيغة العامة | Syntax

المتغيرات:

  • تُعرّف وتُعطى قيمة أولية في قسم DECLARE
  • تُستخدم وتُسند لها قيم جديدة في قسم BEGIN
  • تُمرر كمعاملات (Parameters) للبرامج الفرعية
-- الصيغة العامة:
identifier [CONSTANT] datatype [NOT NULL]
           [:= | DEFAULT expr];

-- أمثلة:
DECLARE
  emp_hiredate    DATE;
  emp_deptno      NUMBER(2) NOT NULL := 10;
  location        VARCHAR2(13) := 'Atlanta';
  c_comm          CONSTANT NUMBER := 1400;

💡 ملاحظة: يمكن إعطاء قيمة أولية بـ := أو بـ DEFAULT: 

Myname VARCHAR2(20) := 'John';
-- أو
Myname VARCHAR2(20) DEFAULT 'John';

💻 أمثلة من المنهج | Examples

مثال 1: قيمة NULL أولاً ثم إسناد

SET SERVEROUTPUT ON
DECLARE
  Myname VARCHAR2(20);
BEGIN
  DBMS_OUTPUT.PUT_LINE('My name is: ' || Myname);  -- NULL
  Myname := 'John';
  DBMS_OUTPUT.PUT_LINE('My name is: ' || Myname);  -- John
END;
/

مثال 2: قيمة أولية ثم تغيير

SET SERVEROUTPUT ON
DECLARE
  Myname VARCHAR2(20) := 'John';
BEGIN
  Myname := 'Steven';
  DBMS_OUTPUT.PUT_LINE('My name is: ' || Myname);  -- Steven
END;
/

🔤 الفواصل في النصوص | Delimiters in String Literals

للتعامل مع نصوص تحتوي على علامة اقتباس مفردة (')، نستخدم q-quoting: 

SET SERVEROUTPUT ON
DECLARE
  event VARCHAR2(15);
BEGIN
  event := q'!Father's day!';
  DBMS_OUTPUT.PUT_LINE('3rd Sunday in June is: ' || event);

  event := q'[Mother's day]';
  DBMS_OUTPUT.PUT_LINE('2nd Sunday in May is: ' || event);
END;
/

💡 الصيغة: q'!text!' أو q'[text]' — أي فاصل (delimiter) يمكن استخدامه بين q' و '

📋 أنواع المتغيرات | Types of Variables

النوع الشرح مثال
Scalar قيمة واحدة (عدد، نص، تاريخ، بولياني) NUMBER, VARCHAR2, DATE, BOOLEAN
Composite مجموعة قيم (سجلات، مصفوفات) RECORD, TABLE, VARRAY
Reference مؤشرات (Pointers) REF CURSOR
LOB كائنات كبيرة (Large Objects) CLOB, BLOB
Bind Variables متغيرات خارجية (Non-PL/SQL) VARIABLE x NUMBER

⚠️ إرشادات التعريف | Guidelines

  • استخدم أسماء ذات معنى (Meaningful Names)
  • أعطِ قيمة أولية للمتغيرات المُعرّفة بـ NOT NULL أو CONSTANT
  • عرّف متغير واحد في كل سطر لتسهيل القراءة
  • تجنّب استخدام أسماء أعمدة الجداول كأسماء متغيرات 
-- ❌ خطأ: اسم المتغير مثل اسم العمود
DECLARE
  employee_id NUMBER(6);
BEGIN
  SELECT employee_id INTO employee_id  -- غموض!
  FROM   employees
  WHERE  last_name = 'Kochhar';
END;
/

أمثلة Scalar Variables (Slide 23)

DECLARE
  emp_job         VARCHAR2(9);

  orderdate       DATE := SYSDATE + 7;
  c_tax_rate      CONSTANT NUMBER(3,2) := 8.25;
  valid           BOOLEAN NOT NULL := TRUE;

📝 نقاط هامة | Key Takeaways

  • الصيغة: identifier [CONSTANT] datatype [NOT NULL] [:= expr]
  • يمكن الإسناد بـ := أو DEFAULT
  • المعرّفات: تبدأ بحرف، حد أقصى 30 حرف، لا كلمات محجوزة
  • 5 أنواع: Scalar, Composite, Reference, LOB, Bind Variable
  • لا تسمِّ متغيراتك بنفس أسماء الأعمدة!
  • q-quoting لكتابة نصوص تحتوي على '
