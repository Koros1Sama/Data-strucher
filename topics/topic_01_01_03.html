Block Types

أنواع كتل PL/SQL

🧩 الأنواع الثلاثة | The Three Types

يمكن تقسيم كتل PL/SQL إلى ثلاثة أنواع رئيسية:

1. كتلة مجهولة (Anonymous Block)

ليس لها اسم، لا تُحفظ في قاعدة البيانات، تُنفذ مرة واحدة فقط ثم تختفي.

للتجربة السريعة

2. إجراء (Procedure)

لها اسم، تُحفظ في قاعدة البيانات، يمكن استدعاؤها مرارًا وتكرارًا. تستخدم لتنفيذ "فعل" (Action).

برنامج محفوظ

3. دالة (Function)

مثل الإجراء، لكنها يجب أن تُرجع قيمة (Return Value). تستخدم عادة للحسابات.

ترجع قيمة

⚖️ مقارنة سريعة | Comparison

النوع الاسم الحفظ في DB إرجاع قيمة
Anonymous لا يوجد ❌ لا ❌ لا
Procedure موجود ✅ نعم اختياري
Function موجود ✅ نعم ✅ نعم (إجباري)

💻 أمثلة للكتابة Syntax

Anonymous Block

[DECLARE]
BEGIN
   -- statements
[EXCEPTION]
END;

Procedure

PROCEDURE name IS
BEGIN
   -- statements
[EXCEPTION]
END;

Function

FUNCTION name RETURN datatype IS
BEGIN
   -- statements
   RETURN value;
[EXCEPTION]
END;
