Bind Variables

متغيرات الربط (Host Variables)

📖 ما هي Bind Variables؟

Bind Variables هي متغيرات:

  • تُنشأ في البيئة الخارجية (Environment) وليس داخل كتلة PL/SQL
  • تُسمى أيضاً Host Variables (متغيرات مضيفة)
  • تُنشأ بكلمة VARIABLE
  • تُستخدم في جمل SQL وكتل PL/SQL
  • يمكن الوصول إليها حتى بعد انتهاء تنفيذ الكتلة
  • تُشار إليها بعلامة نقطتين قبلها (:)

💻 أمثلة من المنهج | Examples

مثال 1: إنشاء واستخدام وطباعة (Slide 27)

-- 1. إنشاء Bind Variable في البيئة:
VARIABLE emp_salary NUMBER

-- 2. استخدامها داخل كتلة PL/SQL:
BEGIN
  SELECT salary INTO :emp_salary
  FROM   employees
  WHERE  employee_id = 178;
END;
/

-- 3. طباعة قيمتها بعد انتهاء الكتلة:
PRINT emp_salary

-- 4. استخدامها في جملة SQL مباشرة:
SELECT first_name, last_name
FROM   employees
WHERE  salary = :emp_salary;

⚠️ ملاحظة: لاحظ علامة : قبل اسم المتغير عند استخدامه — :emp_salary وليس emp_salary

مثال 2: الطباعة التلقائية بـ SET AUTOPRINT ON (Slide 28)

VARIABLE emp_salary NUMBER
SET AUTOPRINT ON

BEGIN
  SELECT salary INTO :emp_salary
  FROM   employees
  WHERE  employee_id = 178;
END;
/
-- القيمة تُطبع تلقائياً بدون الحاجة لـ PRINT

📝 نقاط هامة | Key Takeaways

  • تُنشأ بـ VARIABLE خارج الكتلة (في البيئة)
  • تُشار إليها بـ : (colon) داخل PL/SQL و SQL
  • تبقى متاحة بعد انتهاء الكتلة
  • PRINT لطباعتها يدوياً
  • SET AUTOPRINT ON لطباعتها تلقائياً
