Nested Loops and Labels

الحلقات المتداخلة والتسميات

📖 مقدمة | Introduction

يمكن وضع الحلقات داخل بعضها البعض (Nested Loops) لتنفيذ مهام معقدة. وللتحكم في الخروج من حلقة معينة أو الانتقال إليها، نستخدم التسميات (Labels).

📝 المحتوى | Content

الحلقات المتداخلة (Nested Loops)

هي حلقة داخل حلقة أخرى. الحلقة الداخلية تكمل دورتها كاملة لكل تكرار واحد من الحلقة الخارجية.

التسميات (Labels)

تستخدم لتمييز الكتل البرمجية أو الحلقات. تكتب التسمية بين قوسين مزدوجين <<label_name>>. فائدتها الرئيسية هي السماح بالخروج من حلقات متعددة باستخدام EXIT label_name.

مثال شامل | Comprehensive Example

BEGIN
  <<Outer_Loop>>
  LOOP
    counter := counter + 1;
    EXIT WHEN counter > 10;
    
    <<Inner_Loop>>
    LOOP
      ...
      EXIT Outer_Loop WHEN total_done = 'YES'; -- Exit both loops
      EXIT WHEN inner_done = 'YES';           -- Exit inner loop only
    END LOOP Inner_Loop;
    
  END LOOP Outer_Loop;
END;
