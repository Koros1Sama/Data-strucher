Cursor FOR Loops

حلقات FOR للمؤشرات

📖 مقدمة | Introduction

تعتبر Cursor FOR Loop طريقة مختصرة وفعالة للتعامل مع المؤشرات الصريحة. تقوم هذه الحلقة بفتح المؤشر، وجلب الصفوف واحداً تلو الآخر، ثم إغلاق المؤشر تلقائياً عند انتهاء البيانات، دون الحاجة لكتابة أوامر OPEN، FETCH، و CLOSE يدوياً.

📝 المحتوى | Content

المميزات | Features

  • تقوم بتعريف سجل (Record) ضمنياً %ROWTYPE ليحمل بيانات الصف.
  • تتعامل مع فتح وإغلاق المؤشر تلقائياً (حتى في حالة حدوث خطأ).
  • تجعل الكود أنظف وأسهل للقراءة.

الصيغة العامة | Syntax

FOR record_name IN cursor_name LOOP
  -- Access data using record_name.column_name
  statement1;
  statement2;
END LOOP;

مثال عملي | Practical Example

DECLARE
  CURSOR emp_cursor IS 
    SELECT employee_id, last_name FROM employees WHERE department_id = 30;
BEGIN
  -- لا حاجة لتعريف متغير للسجل، يتم تعريفه تلقائياً
  FOR emp_record IN emp_cursor LOOP
    DBMS_OUTPUT.PUT_LINE( emp_record.employee_id || ' ' || emp_record.last_name );
  END LOOP;
  -- يتم إغلاق المؤشر تلقائياً هنا
END;

استخدام الاستعلام الفرعي مباشرة (Using Subqueries)

يمكننا تعريف المؤشر كجزء من الحلقة مباشرة دون الحاجة لقسم DECLARE: 

BEGIN
  FOR emp_record IN (SELECT employee_id, last_name FROM employees) LOOP
    DBMS_OUTPUT.PUT_LINE(emp_record.last_name);
  END LOOP;
END;
