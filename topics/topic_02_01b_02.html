DML Statements in PL/SQL

جمل معالجة البيانات في PL/SQL

📖 جمل DML | Data Manipulation

يمكنك تعديل بيانات الجداول داخل PL/SQL باستخدام (Slide 31):

  • INSERT — إدراج صفوف جديدة
  • UPDATE — تحديث صفوف موجودة
  • DELETE — حذف صفوف
  • MERGE — دمج (إدراج أو تحديث)

➕ INSERT — إدراج بيانات (Slide 32)

BEGIN
  INSERT INTO employees
    (employee_id, first_name, last_name, email,
     hire_date, job_id, salary)
  VALUES
    (employees_seq.NEXTVAL, 'Ruth', 'Cores',
     'RCORES', SYSDATE, 'AD_ASST', 4000);
END;
/

✏️ UPDATE — تحديث بيانات (Slide 33)

DECLARE
  sal_increase  employees.salary%TYPE := 800;
BEGIN
  UPDATE employees
  SET    salary = salary + sal_increase
  WHERE  job_id = 'ST_CLERK';
END;
/

🗑️ DELETE — حذف بيانات (Slide 34)

DECLARE
  deptno  employees.department_id%TYPE := 10;
BEGIN
  DELETE FROM employees
  WHERE department_id = deptno;
END;
/

📝 نقاط هامة | Key Takeaways

  • INSERT, UPDATE, DELETE, MERGE تعمل داخل PL/SQL كما في SQL
  • يمكن استخدام متغيرات PL/SQL في جمل DML
  • يمكن استخدام Sequences مثل employees_seq.NEXTVAL
  • لا تنسَ COMMIT بعد DML لتثبيت التغييرات
