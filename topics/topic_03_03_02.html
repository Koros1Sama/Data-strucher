Trigger Execution Model — Summary

نموذج تنفيذ المشغّلات — ملخص شامل

📋 ملخص كل ما تعلمناه (Slide 22)

في هذا الدرس تعلمنا:

  • إنشاء مشغّلات قاعدة البيانات (Database Triggers) التي تعمل مع DML
  • إنشاء أنواع Statement و Row triggers
  • استخدام قواعد تنفيذ المشغّلات (Firing Rules)
  • تفعيل وتعطيل وإدارة المشغّلات
  • حذف المشغّلات من قاعدة البيانات

📊 جدول مرجعي شامل

العنصر الخيارات مثال
التوقيت BEFORE | AFTER | INSTEAD OF BEFORE INSERT
الأحداث INSERT | UPDATE [OF col] | DELETE INSERT OR UPDATE
المستوى Statement (افتراضي) | Row (FOR EACH ROW) FOR EACH ROW
القيم :OLD.col | :NEW.col :NEW.salary
المؤشرات INSERTING | UPDATING | DELETING IF DELETING THEN
الشرط WHEN (condition) — بدون : WHEN (NEW.job_id = 'SA_REP')
الإدارة ENABLE | DISABLE | DROP | COMPILE ALTER TRIGGER name DISABLE
