Managing Triggers

إدارة المشغّلات

⚙️ تعطيل وتفعيل (Slide 20)

تعطيل/تفعيل trigger واحد:

ALTER TRIGGER trigger_name DISABLE;
ALTER TRIGGER trigger_name ENABLE;

تعطيل/تفعيل جميع triggers على جدول:

ALTER TABLE table_name DISABLE ALL TRIGGERS;
ALTER TABLE table_name ENABLE ALL TRIGGERS;

إعادة ترجمة trigger:

ALTER TRIGGER trigger_name COMPILE;

🗑️ حذف Trigger (Slide 21)

-- الصيغة:
DROP TRIGGER trigger_name;

-- مثال:
DROP TRIGGER secure_emp;

📌 ملاحظة: جميع triggers على جدول تُحذف تلقائياً عند حذف الجدول نفسه.

⚖️ Triggers مقابل Stored Procedures (Slide 19)

الميزة Trigger Stored Procedure
التنفيذ تلقائي (implicit) يدوي (explicit) بـ EXECUTE
COMMIT لا يمكن استخدام COMMIT/ROLLBACK مباشرة يمكن استخدام COMMIT/ROLLBACK
الاستخدام فرض قواعد تلقائية تنفيذ إجراءات محددة

📝 نقاط هامة | Key Takeaways

  • ALTER TRIGGER ... DISABLE/ENABLE لتعطيل/تفعيل
  • ALTER TABLE ... DISABLE ALL TRIGGERS لجميع triggers
  • DROP TRIGGER لحذف trigger
  • حذف الجدول يحذف جميع triggers عليه تلقائياً
  • Triggers تعمل تلقائياً ≠ Procedures تعمل يدوياً
