Scalar Data Types

أنواع البيانات العددية (المفردة)

🔢 ما هي Scalar Data Types؟

هي أنواع البيانات التي تحمل Single Value (وليست قائمة أو سجل). تنقسم إلى 4 فئات رئيسية:

1. الرقمية (Numeric)

للأرقام الصحيحة والعشرية.

  • NUMBER(p,s): النوع الأشهر.
  • PLS_INTEGER: أسرع للحسابات (صحيح فقط).
  • BINARY_INTEGER: مشابه لـ PLS_INTEGER.

2. النصية (Character)

للنصوص والرموز.

  • VARCHAR2(n): نص متغير الطول (الأكثر استخداماً).
  • CHAR(n): نص ثابت الطول.
  • LONG: نصوص طويلة جداً (قديم).

3. التواريخ (Date/Time)

للتاريخ والوقت.

  • DATE: يخزن التاريخ والوقت بالثانية.
  • TIMESTAMP: أدق (يخزن أجزاء الثانية).

4. المنطقية (Boolean)

للقيم المنطقية.

  • BOOLEAN: يقبل (TRUE, FALSE, NULL).
  • ملاحظة: لا يمكن طباعة القيمة Boolean مباشرة.

💡 تفاصيل NUMBER Data Type

النوع NUMBER(p,s) هو الأكثر مرونة:

  • p (Precision): إجمالي عدد الخانات (Max 38).
  • s (Scale): عدد الخانات بعد الفاصلة.

أمثلة:

  • NUMBER(5,2): يخزن 123.45 (إجمالي 5 خانات، 2 منها عشري).
  • NUMBER(3): يخزن 123 (عدد صحيح).

💻 مثال شامل | Comprehensive Example

DECLARE
    v_emp_id    NUMBER(6) := 1001;
    v_name      VARCHAR2(50) := 'Sarah Ahmed';
    v_salary    NUMBER(8,2) := 5000.50;
    v_active    BOOLEAN := TRUE;
    v_hire_date DATE := '01-JAN-2023';
BEGIN
    DBMS_OUTPUT.PUT_LINE('Employee: ' || v_name);
    DBMS_OUTPUT.PUT_LINE('Salary: ' || v_salary);
    
    -- التعامل مع Boolean
    IF v_active THEN
        DBMS_OUTPUT.PUT_LINE('Status: Active');
    END IF;
END;
