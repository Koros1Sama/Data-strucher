Levels of Isolation

مستويات العزل

📊 المستويات الأربعة (Slide 13)

المستوى الضمانة المشاكل المحلولة
Level 0 لا تكتب فوق Dirty Reads لمعاملات أعلى مستوى —
Level 1 No Lost Updates ✅ Lost Update
Level 2 No Lost Updates + No Dirty Reads ✅ Lost Update + ✅ Dirty Read
Level 3 (True Isolation) Repeatable Reads + كل ضمانات Level 2 ✅ Lost Update + ✅ Dirty Read + ✅ Unrepeatable Read

📸 Snapshot Isolation (Slide 13)

  • بديل شائع لـ True Isolation
  • كل معاملة ترى لقطة (Snapshot) من قاعدة البيانات في لحظة بدايتها
  • لا ترى تغييرات المعاملات الأخرى التي لم تُنهَ بعد

📈 الملخص البصري

Level 0: أدنى حماية
   ↓
Level 1: + No Lost Updates
   ↓
Level 2: + No Dirty Reads
   ↓
Level 3: + Repeatable Reads = True Isolation ✅

📝 نقاط هامة | Key Takeaways

  • 4 مستويات: 0 → 1 → 2 → 3 (True Isolation)
  • Level 1: لا Lost Updates
  • Level 2: + لا Dirty Reads
  • Level 3: + Repeatable Reads (الأكثر أماناً)
  • Snapshot Isolation: بديل عملي شائع
