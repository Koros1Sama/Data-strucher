IF Statements

الجمل الشرطية (IF)

🔀 التحكم في التدفق (Flow Control)

تسمح لك جمل IF بتنفيذ كود معين فقط إذا تحقق شرط محدد.

📝 الأشكال الثلاثة لـ IF

1. IF-THEN (البسيطة)

IF condition THEN
   -- statements to execute if TRUE
END IF;

2. IF-THEN-ELSE (إذا وإلا)

IF condition THEN
   -- execute if TRUE
ELSE
   -- execute if FALSE or NULL
END IF;

3. IF-THEN-ELSIF (شروط متعددة)

لاحظ كتابة ELSIF (بدون E في المنتصف)!

IF condition1 THEN
   statements1;
ELSIF condition2 THEN
   statements2;
ELSE
   statements3;
END IF;

⚠️ قاعدة هامة: التعامل مع NULL

في PL/SQL، الشرط يمكن أن يكون نتيجته: TRUE أو FALSE أو NULL.

  • جملة IF تنفذ الكود فقط إذا كانت النتيجة TRUE.
  • إذا كانت النتيجة FALSE أو NULL، ينتقل التنفيذ إلى ELSE (إن وجدت) أو يخرج من الـ IF.

💻 مثال عملي

DECLARE
    v_salary NUMBER := 15000;
    v_bonus  NUMBER;
BEGIN
    IF v_salary > 20000 THEN
        v_bonus := 1000;
    ELSIF v_salary > 10000 THEN
        v_bonus := 500;
    ELSE
        v_bonus := 100;
    END IF;
    
    DBMS_OUTPUT.PUT_LINE('Bonus: ' || v_bonus);
END;
