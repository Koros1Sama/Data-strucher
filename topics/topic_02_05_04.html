Procedures vs Functions

الفرق بين الإجراءات والدوال

📖 مقارنة سريعة | Quick Comparison

الإجراء (Procedure): ينفذ عملية، قد لا يعيد قيمة، لا يمكن استخدامه في جمل SQL.
الدالة (Function): تحسب وتعيد قيمة واحدة، يمكن استخدامها في جمل SQL (غالباً).

📝 المحتوى | Content

جدول المقارنة | Comparison Table

وجه المقارنة (Aspect) الإجراء (Procedure) الدالة (Function)
القيمة المرجعة (Return Value) اختياري (يمكن استخدام باراميترات OUT) إجباري (Must return a value)
الاستدعاء (Invocation) كجملة مستقلة في كود PL/SQL كجزء من تعبير (Expression) أو جملة SQL
الهدف (Purpose) تنفيذ إجراء معين (Action) مثل تحديث بيانات حساب قيمة وإرجاعها (Computation)
استخدام SQL لا يمكن استدعاؤه داخل جملة SELECT يمكن استدعاؤها داخل جملة SELECT (بشروط)
