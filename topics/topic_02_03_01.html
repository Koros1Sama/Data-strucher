IF Statements

جمل IF الشرطية

📖 الصيغة العامة | Syntax (Slide 40)

IF condition THEN
  statements;
[ELSIF condition THEN
  statements;]
[ELSE
  statements;]
END IF;

💻 مثال: IF ELSIF ELSE (Slide 41)

DECLARE
  myage NUMBER := 21;
BEGIN
  IF myage < 11 THEN
    DBMS_OUTPUT.PUT_LINE('I am a child');
  ELSIF myage < 20 THEN
    DBMS_OUTPUT.PUT_LINE('I am young');
  ELSIF myage < 30 THEN
    DBMS_OUTPUT.PUT_LINE('I am in my twenties');
  ELSIF myage < 40 THEN
    DBMS_OUTPUT.PUT_LINE('I am in my thirties');
  ELSE
    DBMS_OUTPUT.PUT_LINE('I am always young');
  END IF;
END;
/

⚠️ قيم NULL في IF (Slide 42)

DECLARE
  myage NUMBER;   -- قيمته NULL (لم نعطه قيمة)
BEGIN
  IF myage < 11 THEN
    DBMS_OUTPUT.PUT_LINE('I am a child');
  ELSE
    DBMS_OUTPUT.PUT_LINE('I am not a child');
  END IF;
END;
/
-- النتيجة: 'I am not a child'
-- لأن NULL < 11 تُقيَّم كـ NULL (ليس TRUE)، فيذهب لـ ELSE

⚠️ مهم جداً: أي مقارنة مع NULL تُرجع NULL وليس TRUE أو FALSE. لذلك الشرط لا يتحقق وينتقل البرنامج إلى ELSE.

📋 جداول المنطق مع NULL | Logic Tables (Slides 47-48)

  • أي مقارنة بسيطة مع NULL تُرجع NULL
  • NOT NULL يُرجع NULL
  • إذا كان الشرط NULL، لا تُنفَّذ الجمل المرتبطة

📝 نقاط هامة | Key Takeaways

  • IF → ELSIF → ELSE → END IF (لا تنسَ END IF)
  • يمكن تداخل أكثر من ELSIF
  • NULL في الشرط = الشرط لا يتحقق (يذهب لـ ELSE)
  • FALSE AND NULL = FALSE (لأن FALSE يسيطر على AND)
  • TRUE OR NULL = TRUE (لأن TRUE يسيطر على OR)
