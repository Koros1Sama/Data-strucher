Creating Stored Functions

إنشاء الدوال المخزنة

📖 مقدمة | Introduction

الدالة المخزنة (Stored Function) تشبه الإجراء المخزن، لكن الفرق الرئيسي أنها يجب أن تعيد قيمة (RETURN value). تستخدم غالباً لحساب القيم ويمكن استخدامها داخل جمل SQL.

📝 المحتوى | Content

المفهوم | Concept

الدالة المخزنة (Stored Function) تشبه الإجراء المخزن ولكنها يجب أن ترجع قيمة (Return Value). تستخدم غالباً لإجراء العمليات الحسابية أو استرجاع قيمة مفردة.

الصيغة العامة | Syntax

CREATE [OR REPLACE] FUNCTION function_name
    [(parameter_name [IN] type, ...)]
RETURN return_type
IS | AS
    executable_statements
    RETURN expression;
[EXCEPTION
    exception_handlers]
END [function_name];
