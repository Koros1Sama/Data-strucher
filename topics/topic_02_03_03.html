Loop Types: Basic, WHILE, FOR

أنواع الحلقات: أساسية، WHILE، FOR

📖 مقدمة | Introduction

تسمح الحلقات (Loops) بتكرار تنفيذ جملة أو مجموعة من الجمل عدة مرات. يوفر PL/SQL ثلاثة أنواع من الحلقات: Basic Loop، WHILE Loop، و FOR Loop.

📝 المحتوى | Content

1. الحلقة الأساسية (Basic Loop)

تبدأ بكلمة LOOP وتنتهي بـ END LOOP. يتم الخروج منها عادة باستخدام EXIT WHEN. 

LOOP
  -- Statements
  EXIT WHEN condition;
END LOOP;

2. حلقة WHILE (WHILE Loop)

تستمر في التكرار طالما أن الشرط صحيح (TRUE). يتم فحص الشرط في بداية كل دورة. 

WHILE condition LOOP
  statement1;
  statement2;
END LOOP;

3. حلقة FOR (FOR Loop)

تستخدم عندما يكون عدد التكرارات معروفاً مسبقاً. العداد (Counter) يتم تعريفه تلقائياً. 

FOR counter IN [REVERSE] lower..upper LOOP 
  statement1;
  statement2;
END LOOP;

مثال:

FOR i IN 1..3 LOOP
  INSERT INTO locations(...) VALUES(..., i, ...);
END LOOP;
