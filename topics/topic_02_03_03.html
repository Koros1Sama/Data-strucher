Iterative Control: LOOP Statements

الحلقات التكرارية

🔄 أنواع الحلقات | Loop Types (Slide 50)

ثلاثة أنواع:

النوع متى تستخدمها (Slide 58)
Basic LOOP عندما يجب تنفيذ الجمل مرة واحدة على الأقل
WHILE LOOP عندما يجب تقييم الشرط في بداية كل تكرار
FOR LOOP عندما يكون عدد التكرارات معروفاً

1️⃣ Basic LOOP (Slides 51-52)

-- الصيغة:
LOOP
  statement1;
  ...
  EXIT [WHEN condition];
END LOOP;

مثال من المنهج:

DECLARE
  countryid  locations.country_id%TYPE := 'CA';
  loc_id     locations.location_id%TYPE;
  counter    NUMBER(2) := 1;
  new_city   locations.city%TYPE := 'Montreal';
BEGIN
  SELECT MAX(location_id) INTO loc_id FROM locations
  WHERE country_id = countryid;
  LOOP
    INSERT INTO locations(location_id, city, country_id)
    VALUES((loc_id + counter), new_city, countryid);
    counter := counter + 1;
    EXIT WHEN counter > 3;
  END LOOP;
END;
/

2️⃣ WHILE LOOP (Slides 53-54)

-- الصيغة:
WHILE condition LOOP
  statement1;
  statement2;
  ...
END LOOP;

مثال من المنهج:

DECLARE
  countryid  locations.country_id%TYPE := 'CA';
  loc_id     locations.location_id%TYPE;
  new_city   locations.city%TYPE := 'Montreal';
  counter    NUMBER := 1;
BEGIN
  SELECT MAX(location_id) INTO loc_id FROM locations
  WHERE country_id = countryid;
  WHILE counter <= 3 LOOP
    INSERT INTO locations(location_id, city, country_id)
    VALUES((loc_id + counter), new_city, countryid);
    counter := counter + 1;
  END LOOP;
END;
/

3️⃣ FOR LOOP (Slides 55-57)

-- الصيغة:
FOR counter IN [REVERSE]
    lower_bound..upper_bound LOOP
  statement1;
  statement2;
  ...
END LOOP;

مثال من المنهج:

DECLARE
  countryid  locations.country_id%TYPE := 'CA';
  loc_id     locations.location_id%TYPE;
  new_city   locations.city%TYPE := 'Montreal';
BEGIN
  SELECT MAX(location_id) INTO loc_id
  FROM locations WHERE country_id = countryid;
  FOR i IN 1..3 LOOP
    INSERT INTO locations(location_id, city, country_id)
    VALUES((loc_id + i), new_city, countryid);
  END LOOP;
END;
/

📋 إرشادات FOR LOOP (Slide 57):

  • العداد (counter) يُشار إليه فقط داخل الحلقة — غير معرّف خارجها
  • لا يمكن تعيين قيمة للعداد يدوياً (Read-Only)
  • حدود الحلقة (bounds) يجب ألا تكون NULL

📝 نقاط هامة | Key Takeaways

  • Basic LOOP: ينفذ مرة على الأقل، استخدم EXIT WHEN للخروج
  • WHILE LOOP: قد لا ينفذ أبداً إذا الشرط FALSE من البداية
  • FOR LOOP: عدد محدد، العداد تلقائي و Read-Only
  • REVERSE للعد التنازلي في FOR LOOP
