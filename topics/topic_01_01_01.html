What is PL/SQL?

ما هي لغة PL/SQL؟

📖 مقدمة | Introduction

لغة PL/SQL هي اختصار لـ Procedural Language extension to SQL. وهي لغة أنشأتها Oracle لتضيف قدرات البرمجة الإجرائية (مثل الشروط والحلقات) إلى لغة SQL القياسية.

لماذا نحتاجها؟ لغة SQL وحدها لغة غير إجرائية (تطلب البيانات ولا تحدد كيف نحصل عليها). PL/SQL تسمح لك بكتابة منطق برمجي كامل للتعامل مع البيانات.

⚡ المميزات الرئيسية | Key Features

  • تكامل قوي مع SQL: يمكنك كتابة جمل SQL مباشرة داخل كود PL/SQL.
  • أداء عالي: ترسل كتلة الكود كاملة إلى الخادم لتنفيذها مرة واحدة، مما يقلل حركة الشبكة (Network Traffic).
  • الإنتاجية: توفر أدوات قوية للتعامل مع الأخطاء (Exceptions).
  • قابلية النقل: تعمل على أي نظام تشغيل يدعم Oracle Database.

🏗️ البنية | Architecture

يعمل محرك PL/SQL داخل خادم قاعدة البيانات (Database Server) أو أحياناً في أداة التطوير. 

-- مثال بسيط جدًا
BEGIN
   DBMS_OUTPUT.PUT_LINE('Hello, PL/SQL!');
END;

📝 نقاط هامة | Key Takeaways

  • PL/SQL = SQL + Procedural Features (IF, LOOP, etc.)
  • تزيد من كفاءة التطبيقات بتقليل الاتصال بالخادم.
  • هي اللغة القياسية لتطوير تطبيقات قواعد بيانات Oracle.
